Bunpro Toolbox

Userscript Documentation

The script adds options to instantly search the currently viewed grammar point on Bunpro. The following links will be added to the bottom of each grammar page:

Dictionaries, References

  • Jisho (Dictionary)
  • Wikitionary (Dictionary)
  • Eijirou (Example Sentances)

YouTube

  • YouGlish (Provides context sentences detected in Youtube videos)
  • YouTube (Grammar videos)

Q&A, Communities

  • Japanese Stack Exchange (Japanese Q&A website)
  • WaniKani Forum
  • r/LearnJapanese Reddit

These links are shown at the bottom of the page for each grammar point.

This script is very simple to customise, you can add, edit, remove or re-order links.

The userscript will also replace broken links with a archived version.
